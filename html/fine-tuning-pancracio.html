Fine-Tuning de LLMs Locales

ENTRENAMIENTO DE IA: PROYECTO PANCRACIO

Entorno: Windows 11 Nativo con Python 3.10

ÍNDICE DE CONTENIDOS

  1. Preparación del Entorno (Python y Venv)
  2. Estructura de Archivos y Dataset
  3. El Proceso de Entrenamiento (Fine-Tuning)
  4. Pruebas de Inferencia con el Modelo Entrenado

1. PREPARACIÓN DEL ENTORNO

Para realizar esta práctica, utilizamos un equipo con Windows 11 y una tarjeta gráfica RTX 3060 (12GB VRAM). El modelo elegido fue Google Gemma 2 2B debido a su equilibrio entre potencia y ligereza, ideal para hardware de consumo.

Creación del Entorno Virtual

Lo primero fue aislar las dependencias utilizando Python 3.10. Ejecutamos el comando para crear el entorno virtual y posteriormente activarlo.

Comandos utilizados:

py -3.10 -m venv venv para crear.

.\venv\Scripts\activate para activar (fíjate en el indicador (venv) a la izquierda).

2. ESTRUCTURA DE ARCHIVOS Y DATASET

Organización del Proyecto

La carpeta del proyecto contiene los scripts necesarios para el ciclo de vida del modelo:

  • entrenar_wiki_pancracio.py: Script principal de entrenamiento.
  • pancracio_knowledge.json: El dataset con la información.
  • probar_conocimiento.py: Script para chatear con la IA.

El Dataset: "Pancracio"

El objetivo era enseñar a la IA sobre un personaje ficticio: Pancracio, un granjero soriano tradicional. Creamos un archivo JSON con pares de preguntas (instruction) y respuestas (output) definiendo su personalidad, opiniones sobre la tecnología y su vida en Soria.

3. EL PROCESO DE ENTRENAMIENTO (FINE-TUNING)

Utilizamos la librería transformers y peft para realizar un entrenamiento eficiente (LoRA). El script descarga el modelo base de Google y lo adapta con nuestros datos.

Ejecución del Script

Al ejecutar python entrenar_wiki_pancracio.py, el sistema comenzó a procesar las épocas. En la captura se puede observar cómo la "Loss" (pérdida/error) disminuye progresivamente, lo que indica que la IA está aprendiendo.

Finalización y Guardado

Tras completar las épocas configuradas, el modelo se guardó exitosamente en la carpeta local ./pancracio_conocimiento, listo para ser usado.

4. PRUEBAS DE INFERENCIA

Finalmente, utilizamos un script de inferencia interactivo para probar si el modelo realmente había aprendido los nuevos conocimientos.

Código de Prueba

El script probar_conocimiento.py carga el modelo base y le aplica los "adaptadores" entrenados con la información de Pancracio.

Resultados: Chat con la IA

El resultado fue exitoso. Al preguntar "¿Quién es Pancracio?", la IA respondió con la definición exacta del personaje soriano. También respondió coherentemente a preguntas sobre la cosecha y su opinión sobre la gente de ciudad, demostrando que el Fine-Tuning funcionó.
